# Chrome Web Store - Pflichtfelder Checkliste

## Store Listing (Pflichtfelder)

- [ ] **Name** - "Pixel Grid"
- [ ] **Short Description** - Mindestens 1 Zeile
- [ ] **Detailed Description** - Mindestens 1 Absatz
- [ ] **Category** - Muss ausgewählt sein (z.B. "Developer Tools")
- [ ] **Language** - Muss ausgewählt sein (z.B. "English" oder "German")
- [ ] **Screenshots** - Mindestens 1 Screenshot erforderlich (1280x800 oder 640x400)
- [ ] **Icon** - 128x128 Pixel (wird automatisch aus Extension übernommen)

## Privacy & Compliance

- [ ] **Privacy Policy** - Wird oft benötigt, auch wenn keine Daten gesammelt werden
  - Kann eine einfache Seite sein: "This extension stores settings locally only. No data is collected or sent to external servers."
  - Muss öffentlich zugänglich sein (GitHub Pages, eigene Website, etc.)

## Developer Information

- [ ] **Merchant Icon** - 128x128 Pixel (Account-Icon)
- [ ] **Website** (optional, aber empfohlen)
- [ ] **Support URL** (optional, aber empfohlen)

## Häufige Probleme

1. **Privacy Policy fehlt** - Auch wenn keine Daten gesammelt werden, wird oft eine URL benötigt
2. **Screenshots fehlen** - Mindestens 1 Screenshot ist Pflicht
3. **Language nicht ausgewählt** - Muss explizit ausgewählt werden
4. **Category nicht ausgewählt** - Muss explizit ausgewählt werden
